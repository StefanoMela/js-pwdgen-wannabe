# READ ME - WANNABE JS PWD GENERATOR

Consegna dell'esercizio era creare un generatore di password basate su input immessi dall'utente - nome, cognome, colore preferito.

Il compito è stato svolto creando 4 costanti differenti, di modo che le prime tre  catturino i signoli input dell'utente e li visualizzino in vari span riassuntivi delle scelte dell'utente stesso, mentre la quarta è una costante che racchiude in se le altre tre e le stampa a schermo nella sezione dedicata.
